A Mandelbrot/Julia Explorer

Use your ComplexNumber class to generate a dynamic Mandelbrot/Julia Set visualizer using a minimum of 10 distinct colors:

The visualization should use single 800x320 canvas to organize two 400x300 display windows. The left side of the canvas displays the Mandelbrot set, while the right side of the canvas displays Julia sets. The top of the canvas displays the mouse position as a complex number.

User Interaction

Through interaction with the keyboard and mouse a user can change their view of each shape.

Keyboard Interaction

  • Up, Down, Left, Right to recenter/move the Mandelbrot Set.
  • i or I to zoom in on the Mandelbrot Set
  • o or O to zoom out on the Mandelbrot Set
  • r or R to reset to the default view of the Mandelbrot Set
  • Shift+i or Caps Lock+i to zoom in on the Julia Set
  • Shift+o or Caps Lock+o to zoom out on the Julia Set
  • Shift+r or Caps Lock+r to reset to a default view of the Julia Set

Mouse Movement

Mouse movement displays a complex number centered the top of your canvas, which is determined by mapping the cursor position to current zoom/centering attributes for the appropriate display window.

Mouse Press

Clicking on the left side of the canvas selects a point to generate a specific Julia Set on the right side of the canvas.

Clicking on the right side of the canvas re-centers the current Julia set around the point that was clicked.

Architectural Details

This project will utilize four different Java files:

File Name Purpose
JuliaSetGenerator.java Handles the animation loop, recursive formulas, user interaction.
DisplayWindow.java A data type for working with multiple display windows.
ComplexNumber.java A data type for working with complex numbers.
StdDraw.java A library for graphics and user interaction.
